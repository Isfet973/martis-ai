Crie uma API REST em Python com FastAPI para um sistema de autenticacao JWT com refresh token.
C
from fastapi import FastAPI, Depends, HTTPException from fastapi.security import OAuth2PasswordBearer from jose import JWTError, jwt from datetime import datetime, timedelta from passlib.context import CryptContext SECRET_KEY = "sua-chave-secreta" ALGORITHM = "HS256" app = FastAPI() pwd_ctx = CryptContext(schemes=["bcrypt"]) oauth2 = OAuth2PasswordBearer(tokenUrl="login") def create_token(data: dict, expires: timedelta): payload = data.copy() payload["exp"] = datetime.utcnow() + expires return jwt.encode(payload, SECRET_KEY, ALGORITHM) @app.post("/login") def login(username: str, password: str): # validar usuario no banco aqui access = create_token({"sub": username}, timedelta(minutes=15)) refresh = create_token({"sub": username}, timedelta(days=7)) return {"access_token": access, "refresh_token": refresh} @app.post("/refresh") def refresh(token: str): try: data = jwt.decode(token, SECRET_KEY, [ALGORITHM]) new_access = create_token({"sub": data["sub"]}, timedelta(minutes=15)) return {"access_token": new_access} except JWTError: raise HTTPException(401, "Token invalido")
✓ Access token (15min) + Refresh token (7 dias) + Validacao automatica

Martis-Think Exemplo real
U
Uma bola e lancada verticalmente com velocidade inicial de 20 m/s. Qual a altura maxima e o tempo total de voo? (g = 10 m/s²)
T
Raciocinio passo a passo:

Etapa 1 — Altura maxima
Na altura maxima, a velocidade e zero. Uso a equacao:
v² = v₀² − 2g·h
0 = 20² − 2 × 10 × h
h = 400 / 20 = 20 metros

Etapa 2 — Tempo para atingir o pico
v = v₀ − g·t → 0 = 20 − 10t → t_subida = 2s

Etapa 3 — Tempo total
Por simetria, o tempo de descida = tempo de subida.
t_total = 2 × 2s = 4 segundos

✓ Altura maxima: 20 m  |  Tempo de voo: 4 s
