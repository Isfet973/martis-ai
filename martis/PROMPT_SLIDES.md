# Prompt para Geracao de Slides - Martis AI

Use este prompt em uma IA (ChatGPT, Claude, Gemini, etc.) para gerar uma apresentacao de slides sobre o projeto Martis AI.

---

## PROMPT (copie e cole na IA):

```
Crie uma apresentacao de slides profissional sobre o projeto "Martis AI" com as seguintes caracteristicas:

## INFORMACOES DO PROJETO:

**Nome:** Martis AI
**Tipo:** [AJUSTE: Hub de modelos de IA / Plataforma de IA / Agregador de modelos de IA]
**Slogan:** "Onde deuses teciam redes, nos tecemos inteligencia"
**Inspiracao:** Britomartis, deusa grega das redes de pesca

**O que e:**
- Uma plataforma que centraliza acesso a multiplos modelos de IA
- Interface unica para conversar com GPT-4, Claude, Gemini e +400 modelos
- Sem necessidade de multiplas assinaturas ou chaves de API
- Funciona 100% no navegador, sem instalacao

**Modelos disponiveis:**
- Martis-70B: Raciocinio profundo e analise
- Martis-Summary: Resumo inteligente de textos
- Martis-Code: Programacao em 40+ linguagens
- Martis-Think: Matematica e logica passo a passo
- Martis-Write: Escrita criativa e copywriting
- Martis-Vision: Analise de imagens e OCR
- Martis-Voice: Transcricao e sintese de voz

## VANTAGENS ECONOMICAS:
[AJUSTE os valores conforme necessario]

1. **Reducao de custos com assinaturas:**
   - ChatGPT Plus: $20/mes
   - Claude Pro: $20/mes
   - Gemini Advanced: $20/mes
   - Total economizado: ate $60/mes por usuario

2. **Centralizacao de ferramentas:**
   - Uma unica plataforma substitui 5-10 ferramentas
   - Reducao de tempo gasto alternando entre servicos
   - Curva de aprendizado unica

3. **Escalabilidade:**
   - API gratuita (Puter.js) sem limites
   - Sem custos de infraestrutura
   - Implementacao em minutos

## INOVACOES TECNICAS:

1. **Zero configuracao:**
   - Nao precisa de chaves de API
   - Nao precisa de backend proprio
   - Funciona direto no navegador

2. **API unificada:**
   - Acesso a +400 modelos com uma unica integracao
   - Troca de modelo com uma linha de codigo
   - Fallback automatico entre provedores

3. **Experiencia do usuario:**
   - Status de conexao em tempo real
   - Interface bilingue (PT/EN)
   - Tema claro/escuro
   - Design responsivo

4. **Privacidade:**
   - Modelos podem rodar localmente
   - Sem armazenamento de dados em servidores
   - Codigo aberto e auditavel

## PUBLICO-ALVO:
[AJUSTE conforme seu caso]
- Desenvolvedores que precisam de multiplas IAs
- Empresas que querem reduzir custos com IA
- Estudantes e pesquisadores
- Profissionais de conteudo e marketing

## PROXIMOS PASSOS:
[AJUSTE conforme seu planejamento]
- Adicionar mais modelos especializados
- Implementar sistema de plugins
- Criar marketplace de prompts
- Desenvolver versao mobile

## FORMATO DA APRESENTACAO:

- [AJUSTE: 10-15 slides / 5-8 slides / 20+ slides]
- Estilo visual: [AJUSTE: Minimalista tech / Corporativo / Startup moderno]
- Tom: [AJUSTE: Profissional / Casual / Academico]
- Incluir: [AJUSTE: Graficos comparativos / Screenshots / Fluxogramas]

## ESTRUTURA SUGERIDA:

1. Slide de abertura (nome + slogan)
2. O problema (fragmentacao de ferramentas de IA)
3. A solucao (Martis AI)
4. Como funciona (demo visual)
5. Modelos disponiveis
6. Vantagens economicas
7. Inovacoes tecnicas
8. Comparativo com concorrentes
9. Casos de uso
10. Proximos passos
11. Slide de fechamento (CTA)

Por favor, gere o conteudo completo para cada slide com:
- Titulo
- Bullets principais (maximo 4-5 por slide)
- Notas do apresentador (o que falar em cada slide)
- Sugestao de visual/imagem para cada slide
```

---

## COMO USAR:

1. Copie o prompt acima
2. Ajuste as partes marcadas com [AJUSTE: ...]
3. Cole em uma IA de sua preferencia (ChatGPT, Claude, Gemini)
4. Use o conteudo gerado para criar os slides no PowerPoint, Google Slides ou Canva

---

## DICAS:

- **Para apresentacao academica:** Enfatize as inovacoes tecnicas e cite referencias
- **Para apresentacao de startup:** Foque nas vantagens economicas e escalabilidade
- **Para apresentacao tecnica:** Inclua diagramas de arquitetura e exemplos de codigo
- **Para apresentacao comercial:** Destaque ROI, casos de uso e depoimentos
