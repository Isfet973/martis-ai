Quick Start

In less than 5 minutes you can be chatting with a Martis model — directly in the browser or via API.

Option 1 — Use in the browser

No setup required. Go to martis.ai, click 'Start Free' and choose a model.

Option 2 — Via API (curl)

SHELL 
curl https://api.martis.ai/v1/chat/completions \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "martis-70b-instruct",
    "messages": [{"role": "user", "content": "Hello!"}]
  }'

Option 3 — Python SDK

PYTHON 
from openai import OpenAI

client = OpenAI(
    base_url="https://api.martis.ai/v1",
    api_key="YOUR_API_KEY"
)

resp = client.chat.completions.create(
    model="martis-70b-instruct",
    messages=[{"role": "user", "content": "Hello!"}]
)

print(resp.choices[0].message.content)
✅ OpenAI-compatible — The Martis API is compatible with the OpenAI format. If you already use the OpenAI SDK, just swap base_url and api_key.

Authentication

All API requests require an API key in the Authorization header.

HTTP HEADER
Authorization: Bearer mk_live_xxxxxxxxxxxxxxxxxxxx
⚠️ Never expose your API key in front-end code or public repositories. Use environment variables or a server-side proxy.

API Reference

Endpoints

Method Endpoint Description
POST /v1/chat/completions Generate a chat response
GET /v1/models List all available models
POST /v1/embeddings Generate text embeddings
POST /v1/images/generate Generate images (Martis-Art)
POST /v1/audio/transcriptions Transcribe audio to text (Martis-Voice)

Streaming

To receive responses in real time, add stream: true to the request.

PYTHON — STREAMING
stream = client.chat.completions.create(
    model="martis-70b-instruct",
    messages=[{"role": "user", "content": "Write a poem about the sea"}],
    stream=True
)

for chunk in stream:
    delta = chunk.choices[0].delta.content
    if delta:
        print(delta, end="", flush=True)

Free AI APIs for Real-Time Testing

To add real AI to your Martis project at zero cost, use Google Gemini for text and Hugging Face for images. Both work directly from the browser, no backend needed.

1. Google Gemini Flash — Text AI

ℹ️ Why Gemini? — It's the fastest free text model available today. The free tier includes 15 requests per minute and 1 million tokens per day — more than enough for demos.

Step 1 — Get the API Key

  1. Go to aistudio.google.com
  2. Sign in with your Google account
  3. Click Get API Key → Create API Key
  4. Copy the key — it starts with AIza...

Step 2 — Call the API in HTML

JAVASCRIPT
const GEMINI_KEY = 'YOUR_API_KEY_HERE';

async function askGemini(prompt) {
  const res = await fetch(
    `https://generativelanguage.googleapis.com/v1beta/models/gemini-1.5-flash:generateContent?key=${GEMINI_KEY}`,
    {
      method: 'POST',
      headers: { 'Content-Type': 'application/json' },
      body: JSON.stringify({
        contents: [{ parts: [{ text: prompt }] }]
      })
    }
  );
  const data = await res.json();
  return data.candidates[0].content.parts[0].text;
}

// Usage:
const reply = await askGemini("Explain quantum computing simply");
document.getElementById('output').textContent = reply;

With conversation history (multi-turn)

JAVASCRIPT — MULTI-TURN
let history = [];

async function chat(userMessage) {
  history.push({ role: 'user', parts: [{ text: userMessage }] });

  const res = await fetch(
    `https://generativelanguage.googleapis.com/v1beta/models/gemini-1.5-flash:generateContent?key=${GEMINI_KEY}`,
    {
      method: 'POST',
      headers: { 'Content-Type': 'application/json' },
      body: JSON.stringify({ contents: history })
    }
  );

  const data = await res.json();
  const reply = data.candidates[0].content.parts[0].text;

  history.push({ role: 'model', parts: [{ text: reply }] });
  return reply;
}
⚠️ The Gemini API key can be called directly from the browser for learning projects. For production, use a server-side proxy to hide the key.

2. Hugging Face — Image Generation (Free)

ℹ️ Why Hugging Face? — Hugging Face's free tier gives access to Stable Diffusion XL and other image models directly via REST API, with no server of your own.

Step 1 — Get the Token

  1. Create a free account at huggingface.co
  2. Go to Settings → Access Tokens → New Token
  3. Choose Read as permission and copy the token

Step 2 — Generate image in HTML

JAVASCRIPT
const HF_TOKEN = 'hf_YOUR_TOKEN_HERE';

async function generateImage(prompt) {
  const res = await fetch(
    'https://api-inference.huggingface.co/models/stabilityai/stable-diffusion-xl-base-1.0',
    {
      method: 'POST',
      headers: { 'Authorization': `Bearer ${HF_TOKEN}` },
      body: JSON.stringify({ inputs: prompt })
    }
  );

  const blob = await res.blob();          // response is raw image bytes
  const url  = URL.createObjectURL(blob);  // create a local image URL
  document.getElementById('result-img').src = url;
}

// Usage:
await generateImage("A fishing net made of glowing blue light, digital art");
💡 The first call may take 20–30 seconds while the model loads on Hugging Face's servers. Subsequent calls are much faster.

Combining both APIs in one project

You can use Gemini to generate a creative image prompt and then send that prompt to Hugging Face to generate the image — all in one connected experience.

JAVASCRIPT — FULL FLOW
async function textToImageFlow(userIdea) {

  // Step 1: Ask Gemini to turn the user's idea into an image prompt
  const imagePrompt = await askGemini(
    `Turn this idea into a detailed image generation prompt for Stable Diffusion.
     Only return the prompt, nothing else.
     Idea: "${userIdea}"`
  );

  // Step 2: Send that prompt to Hugging Face to generate the image
  await generateImage(imagePrompt);

  return imagePrompt; // optionally show the prompt used
}

// Example:
await textToImageFlow("A fisherman goddess on the internet");
// Gemini generates: "Ancient Greek goddess made of glowing network cables,
// holding a digital fishing net, ethereal blue light, 8k, photorealistic"
// Hugging Face renders that image automatically
✅ Setup summary: Text → Gemini Flash (free, direct browser call, 1M tokens/day). Image → Hugging Face SDXL (free, no server). Zero cost, zero backend.

Martis CLI

The command-line interface for managing models locally.

Installation

SHELL
# macOS / Linux
curl -fsSL https://martis.ai/install.sh | bash

# Windows (PowerShell admin)
irm https://martis.ai/install.ps1 | iex

Commands

CommandDescription
martis pull <model>Download a model from the repository
martis run <model>Start the model in terminal
martis run <model> --apiExpose the model as local REST API
martis listList downloaded models
martis remove <model>Remove a local model
martis updateUpdate the CLI

Frameworks

LangChain

PYTHON
from langchain_openai import ChatOpenAI

llm = ChatOpenAI(
    model="martis-70b-instruct",
    base_url="https://api.martis.ai/v1",
    api_key="YOUR_API_KEY"
)

response = llm.invoke("Summarize the history of the internet")
print(response.content)
