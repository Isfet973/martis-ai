What is the fundamental difference between deep learning and traditional machine learning?
7
Deep learning is a subset of ML, but the difference goes deeper than hierarchy.

Traditional machine learning requires engineers to manually extract and select features — you decide which data characteristics matter and hand them to the algorithm.

Deep learning uses multi-layer neural networks to learn features automatically from raw data. The network builds hierarchical representations: pixels → edges → shapes → objects.

Practical result: Traditional ML works well with small data and is interpretable. Deep learning excels with large unstructured data (images, audio, text) but requires far more compute and is harder to interpret.
Installation guide

How to download and run locally

Follow the steps below to run Martis-70B Instruct on your machine. Requirement: 16 GB RAM · 12 GB VRAM recommended.

01

Install Martis CLI

Martis CLI works on Windows, macOS and Linux.

# macOS / Linux curl -fsSL https://martis.ai/install.sh | bash # Windows (PowerShell) irm https://martis.ai/install.ps1 | iex # Verify install martis --version
02

Download the model

martis pull martis-70b-instruct # Output: # ✓ Connecting to repository… # → martis-70b-instruct.gguf # ████████████░░░░ 63% · 41 GB # ↳ Verifying integrity… ✓
💾 Size: 41 GB · Q4_K_M
03

Run the model

# Interactive terminal chat martis run martis-70b-instruct # Or as a local REST API (OpenAI-compatible) martis run martis-70b-instruct --api --port 11434 # Test the API curl http://localhost:11434/v1/chat/completions \ -H "Content-Type: application/json" \ -d '{"model":"martis-70b-instruct","messages":[{"role":"user","content":"Hello!"}]}'
04

Integrate into your project

from openai import OpenAI client = OpenAI( base_url="http://localhost:11434/v1", api_key="martis-local" # any string ) resp = client.chat.completions.create( model="martis-70b-instruct", messages=[{"role": "user", "content": "Hello, Martis!"}] ) print(resp.choices[0].message.content)
