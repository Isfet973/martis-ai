What it does

What Martis-Voice 3B does

Martis-Voice specializes in audio: converts speech to text (STT) with speaker diarization, synthesizes natural voice in 15 languages (TTS) and detects emotional sentiment in conversations. Works in real time even on modest hardware.

Installation guide

How to download and run locally

Follow the steps below to run Martis-Voice 3B on your machine. Requirement: 4 GB RAM · CPU (GPU optional).

01

Install Martis CLI

Martis CLI works on Windows, macOS and Linux.

# macOS / Linux curl -fsSL https://martis.ai/install.sh | bash # Windows (PowerShell) irm https://martis.ai/install.ps1 | iex # Verify install martis --version
02

Download the model

martis pull martis-voice-3b # Output: # ✓ Connecting to repository… # → martis-voice-3b.gguf # ████████████░░░░ 63% · 1.8 GB # ↳ Verifying integrity… ✓
💾 Size: 1.8 GB · F16
03

Run the model

# Interactive terminal chat martis run martis-voice-3b # Or as a local REST API (OpenAI-compatible) martis run martis-voice-3b --api --port 11434 # Test the API curl http://localhost:11434/v1/chat/completions \ -H "Content-Type: application/json" \ -d '{"model":"martis-voice-3b","messages":[{"role":"user","content":"Hello!"}]}'
04

Integrate into your project

from openai import OpenAI client = OpenAI( base_url="http://localhost:11434/v1", api_key="martis-local" # any string ) resp = client.chat.completions.create( model="martis-voice-3b", messages=[{"role": "user", "content": "Hello, Martis!"}] ) print(resp.choices[0].message.content)
