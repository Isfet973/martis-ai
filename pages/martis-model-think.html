What Martis-Think 13B does

Real example

See it in action

Martis-Think 13B — Sample conversation
U
How many trailing zeros does 100! (factorial of 100) have?
1
To count trailing zeros in 100!, I count factors of 10 = 2 × 5. Since factors of 2 are far more abundant, I only need to count factors of 5.

Step 1 — Multiples of 5 up to 100:
⌊100/5⌋ = 20

Step 2 — Multiples of 25 (contribute 2 factors of 5):
⌊100/25⌋ = 4

Step 3 — Multiples of 125: ⌊100/125⌋ = 0

Total: 20 + 4 = 24 trailing zeros in 100!
Installation guide

How to download and run locally

Follow the steps below to run Martis-Think 13B on your machine. Requirement: 8 GB RAM · 6 GB VRAM recommended.

01

Install Martis CLI

Martis CLI works on Windows, macOS and Linux.

# macOS / Linux curl -fsSL https://martis.ai/install.sh | bash # Windows (PowerShell) irm https://martis.ai/install.ps1 | iex # Verify install martis --version
02

Download the model

martis pull martis-think-13b # Output: # ✓ Connecting to repository… # → martis-think-13b.gguf # ████████████░░░░ 63% · 8.5 GB # ↳ Verifying integrity… ✓
💾 Size: 8.5 GB · Q5_K
03

Run the model

# Interactive terminal chat martis run martis-think-13b # Or as a local REST API (OpenAI-compatible) martis run martis-think-13b --api --port 11434 # Test the API curl http://localhost:11434/v1/chat/completions \ -H "Content-Type: application/json" \ -d '{"model":"martis-think-13b","messages":[{"role":"user","content":"Hello!"}]}'
04

Integrate into your project

from openai import OpenAI client = OpenAI( base_url="http://localhost:11434/v1", api_key="martis-local" # any string ) resp = client.chat.completions.create( model="martis-think-13b", messages=[{"role": "user", "content": "Hello, Martis!"}] ) print(resp.choices[0].message.content)
