Installation guide

How to download and run locally

Follow the steps below to run Martis-Vision 13B on your machine. Requirement: 8 GB RAM · 8 GB VRAM recommended.

01

Install Martis CLI

Martis CLI works on Windows, macOS and Linux.

# macOS / Linux curl -fsSL https://martis.ai/install.sh | bash # Windows (PowerShell) irm https://martis.ai/install.ps1 | iex # Verify install martis --version
02

Download the model

martis pull martis-vision-13b # Output: # ✓ Connecting to repository… # → martis-vision-13b.gguf # ████████████░░░░ 63% · 9.2 GB # ↳ Verifying integrity… ✓
💾 Size: 9.2 GB · Multimodal
03

Run the model

# Interactive terminal chat martis run martis-vision-13b # Or as a local REST API (OpenAI-compatible) martis run martis-vision-13b --api --port 11434 # Test the API curl http://localhost:11434/v1/chat/completions \ -H "Content-Type: application/json" \ -d '{"model":"martis-vision-13b","messages":[{"role":"user","content":"Hello!"}]}'
04

Integrate into your project

from openai import OpenAI client = OpenAI( base_url="http://localhost:11434/v1", api_key="martis-local" # any string ) resp = client.chat.completions.create( model="martis-vision-13b", messages=[{"role": "user", "content": "Hello, Martis!"}] ) print(resp.choices[0].message.content)
